Doctor Recommendation System via RAG and Vector Search

Objective: Developed an intelligent recommendation system leveraging Retrieval-Augmented Generation (RAG) and vector similarity search.

Scraped and extracted extensive data regarding doctors and their specialties from medical websites using Scrapy and Selenium .
Preprocessed the acquired data and generated robust text embeddings for the extracted information.
Stored and retrieved vector embeddings efficiently utilizing the FAISS vector database .
Implemented a RAG architecture to accurately match patients’ symptoms with appropriate medical profiles.
Developed a semantic search mechanism to recommend the most suitable doctors based on vector similarity.
